### ✅ Характеристики
- **CPU:** Core Ultra 7 / i7-1355U / Ryzen 7 7730U
- **RAM:** 16 ГБ LPDDR5
- **SSD:** 512 ГБ NVMe PCIe 4.0
- **Дисплей:** 14" OLED 2.8K (2880×1800), 90 Гц, 16:10
- **Автономность:** 8-10 часов (混合 режим)
- **Вес:** 1.5 кг

### 🎨 Почему идеально
- **OLED-экран** — лучший для дизайна и кодинга (цвета, контраст)
- **16:10 формат** — больше места для кода/текстов
- **90 Гц** — плавность при свайпах
- **14"** — легко носить в сумке
- **Core Ultra / i7 H-серии** — комфортная скорость для IDE

### ⚠️ Минусы
- Глянец OLED — может бликовать на солнце
- Нет оптического диска
- 16 ГБ RAM распаяна (не апгрейд)
