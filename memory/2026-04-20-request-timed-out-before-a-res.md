## 🔄 Через веб-чат (webchat)

На данный момент **нет прямой кнопки**, но можно:

1. **Отправить сообщение с переключением**:
   ```
   /model ollama/qwen3.5
   ```
   или
   ```
   /model claude-sonnet-4.6
   ```

2. **Или через команду в чате** (если поддерживается):
   ```
   model: ollama/qwen3.5
   ```

## 📝 Через терминал (CLI)

Если используешь OpenClaw через CLI:

```bash
# Переключение на Qwen 3.5
openclaw model ollama/qwen3.5

# Или переключение на Claude (через Bedrock)
openclaw model bedrock:anthropic/claude-sonnet-4.6
```

## ⚙️ Через конфиг-файл

Если хочешь установить модель по умолчанию:

1. Открой `~/.openclaw/config.yml`:
   ```yaml
   model:
     default: ollama/qwen3.5
   ```

2. Сохрани и запусти `openclaw gateway start`

## 📌 Быстрая справка

- **/model [name]** — переключить модель через чат
- **openclaw model [model]** — через CLI
- **~/.openclaw/config.yml** — для установки дефолта
